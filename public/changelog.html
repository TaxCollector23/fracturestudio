Product Changelog

What is new in
Fracture Studio.

A running record of the improvements you can see and use: clearer reports, more practical writing tools, and better ways to return to your work.

June 2, 2026
Final usability pass

Readable reports, clearer navigation, and first-time setup

This update makes Fracture Studio easier to understand on first use and more useful after an analysis is complete.

  • First-time onboarding. New users can choose how they want to start, sign in, continue as a guest, and tell Fracture what kind of work they are preparing.
  • Readable live reports. Studio now types the human report into the same place the final review appears, while raw engine structure stays hidden.
  • Better report controls. Copy the readable report, download it as text, print it, or export a cleaner PDF report.
  • More usable layout. The main Studio workspace is larger, with the draft and final review side by side on desktop and stacked cleanly on mobile.
  • New information pages. About, Credits, Contact, and Onboarding are now part of the core site map.
June 2, 2026
Workspace update

A clearer path back to earlier work

Saved arguments now have a dedicated workspace, making it easier to return to a draft and understand what needs attention before the next revision.

  • Dedicated Past Work page. Open saved arguments as readable cards with draft previews and Fracture-generated report summaries.
  • Faster return to Studio. Reopen an earlier draft or send it directly into rebuttal preparation without pasting it again.
  • More readable navigation. Brand text, supporting labels, and compact interface copy are easier to scan across the site.
June 1, 2026
Release 2.3

Reports that show how an argument holds together

Fracture reports now connect individual weak points to the larger argument, so the next revision is easier to choose and easier to complete.

  • Dependency view. See how a supporting point connects to the conclusion and what loses force if that connection breaks.
  • Dedicated rebuttals workspace. Open current or saved Studio work directly in Rebuttals for strategic opponent attacks, practical answers, crossfire questions, and weighing lines.
  • Clearer source review. Quoted passages receive a public-web check, and tentative matches stay clearly labeled until you confirm the source.
  • More readable exports. PDF reports include the argument network, higher-impact fixes, and a cleaner revision path.
June 1, 2026
Saved work

Work history you can actually revisit

Settings now makes saved work easier to scan and more useful when you are returning to an argument after a break.

  • Expandable saved arguments. Open a history item to review the original argument without leaving Settings.
  • Report summaries in context. Saved analyses show the overall score, verdict, and top revision priority when available.
  • Clear guest expectations. Guest drafts are temporary, while signed-in work can be revisited from cloud history.
June 1, 2026
Studio workflow

A more focused Studio workspace

The Studio now keeps analysis work, follow-up questions, and evidence checks in a cleaner review flow.

  • Clear audit progress. A paced status line and visible percentage show how the full audit is moving without distracting from the finished report.
  • Argument map. See the thesis, supporting claims, and the most vulnerable pressure point in a compact visual structure.
  • Focused follow-up coaching. Ask Fracture Chat about a mapped pressure point in a wider conversation area directly above the source review.
  • Cleaner report flow. Works Cited now stays at the bottom of Studio so it never interrupts the main reasoning analysis.
  • Completion notice. Keep reviewing another page while an audit finishes, then return to Studio from the report-ready notification.
June 1, 2026
Reports

Reports built for revision

Fracture reports now guide the next edit more directly, with clearer structure and stronger attention to evidence.

  • Prioritized revision moves. Reports surface the highest-impact fixes first and connect them to exact wording.
  • Source verification. Evidence-heavy drafts receive a citation-focused review alongside the argument analysis.
  • Export-ready reports. Download a polished PDF or keep the report for a later pass.
